The Garmin should be as near 100% accurate as it's possible to get.....but.... it is measuring the distance travelled in the time since it last updated rather than measuring in real time so is showing an average speed not actual. The only way to get it spot on is to ride at a constant speed for a few seconds and see what the Zumo says.

You should then be able to calibrate your Yellow Box from the readings then go out and try again to make sure you got it right.

And don't forget your van's speedo is quite likely to be out as much as 10% if it's on standard tyres - more if they are a different size and/or worn.
