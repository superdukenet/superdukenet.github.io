I spent the training showing my dad the best line and some braking points.

Second place behind the champion of 2009 - Dirk Schnieders
It was okay

Especially because I knew that he used new soft tyres front and rear.
And not 2 days old soft front and endurance rear like me

For the second qualification I changed the transmission (? correct word?)
Dirk uses 16-39 or 16-38, not sure.
Im using 17-37 - works better for me.

After both qualifications the start order was set and it was time to change the tyres

I dont want to start with old tyres in a race

Soft front, soft rear. I was ready to attack
