I think that it's unwise to run a map for a standard exhaust with a CAT and expect it to be OK for an unrestricted silencer. If you load the Akra map onto the ECU then I'm pretty sure it would eventually damage the CAT when you ran it with the standard set up.

You could fit a PC111 and have the richer map on that and then disconnect it and run on the standard ECU map when you change over your exhaust set up?

There are a lot of people here who know more than me on this subject, they will be along to advise you correctly.
