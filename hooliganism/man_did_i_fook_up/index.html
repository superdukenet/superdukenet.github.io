+1 on above!

Start with disconnecting the battery again and actually check if there is juice left in it (put a 12 v testlight over it) ! Since it is a Lifopho it might be completely discharged.
If it is fine put it back in and makie sure it is the right way around now!
Then use the 12 v test light to follow the wiring and see where the issue is, not that hard. Put the negative clamp if the test light on the engine and start with the light at the battery and follow the cables over the main relay, fuses etc.
As mentioned above it is possibly the 30A fuse.
