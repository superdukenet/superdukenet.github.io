Bic, it works better on the mac than it does on the PC, for some reason quicktime on the PC skips a couple frames every so many frames, mine used to play videos on the PC ok through vlc player but even that seems to be glitchy now, must have had an update or something, mines not the latest tech but is a powerful gaming laptop so performance shouldn't be an issue which it can be on older PCs

With the mac though it ran straight off the bat through quicktime and imovie is excellent for editing the footage ready for youtube, I usually use Streamclip to chop out the main portion I want first as it opens instantly and then load the shortened version into imovie as this takes time for it to process so the smaller file the quicker it is.

Streamclip also makes it fairly easy to work out your lap times.

The motorsports kit which is priced for in the first post comes with various adhesive mounts and a suction mount but if you're not running a race mask to stick one of these on then you'll probably want to buy the handlebar mount for £20 as well which is what most of us have used.

2011-06-01 08:32:00 UTC

just a heads up for people ordering the LCD bacpac ... you need to update the GoPro's firmware as well from here , there are video's explaining how to on there as well.
If you don't update the firmware the lcd will just produce some crackling noise from the speaker but show no image. This might cause some slight panic if you're like me and just pop it on without reading the manual first but it works fine after the firmware update.
