i would be very tempted to drill out the lever ends and put ball joints on there, make up a linkage with a socket on each end, the type that lock in place with a snap clip, get a lh thread and a rh thread and a threaded rod with nuts and set it up just like a gear linkage rod. my ducati's use that system as standard and would be better than trying to straighten your current linkage, the throttle shafts run in only bushes so if it's even slightly skewiff it will wear the bush prematurely.
