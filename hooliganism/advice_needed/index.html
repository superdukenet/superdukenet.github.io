the hard way to block the holes is to tap a thread and cut bolts to length and use a thread locker (MY WAY).... the easy way is to use plastic plugs and glue


when you connect the bike to tune ecu.. you have to download the map from the bike make your changes, save it, then load it back on the bike... bike has to been on with the run switch on.... for the 07 remove the headlight fuse so you dont drain your battery

dont need to load a different map but a dyno run after you have done all the mods is a good move
