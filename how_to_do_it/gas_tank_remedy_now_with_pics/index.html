Just so everyone knows, the big green thing labelled "my little friend" is not a drill press; it is a vertical milling machine. I didn't drill the holes into the bracket; it elongated them using a side cutting end mill, not a drill bit. I don't think a drill press will work for this; the next best tool is going to be a dremel tool or die grinder with an appropriate sized grinding stone. Don't want anyone fecking up their brackets using the wrong tool.
