Sounds like you need a new water pump seal (ktmtwins.com sells kits). It's a somewhat tedious process but not overly complicated, send the SO out shopping for a bit so you can have rights to the oven. Search advrider or the must read links on this site for instructions with lots of pictures. If you can do an oil change on this engine, you can probably do a WP seal.

As for the leaking, is there a puddle of oil anywhere? If not, clean it up, try and determine exactly where the leak is coming from. If it is the left engine case cover (likely the "inner"), loosen up all the bolts by a 1/4 turn or so in a criss cross pattern, then snug them in the same fashion, not too tight though, about an 1/8 turn past the stop point. Criss cross each bolt to the stop point, then go back in the same order torquing each one 1/8 turn. Once the residual oil evacuates the crevices you should be running clean.
