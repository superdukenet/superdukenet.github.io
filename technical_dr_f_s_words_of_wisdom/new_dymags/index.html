TLS_Russ.

If you put the Braketech discs on and elect for the cast iron rotors make sure you use a set of old pads to scrub off the black protective coating. I'm still going through the learning curve on this because I didn't have a spare set, used some new ones that then developed a sort of judder that came back through the lever. Advised that it was contaminated pads so I cleaned them off and sanded back to a clean finish and refitted. Problem seems to have gone but they are still bedding in again and only time will tell - it may be that the guncote has penetrated deep into the pad material. Anyway, just a warning not to ignore the Braketech recommendation as I did. (I don't suppose this is an issue with stainless rotors if they don't Guncote them.
