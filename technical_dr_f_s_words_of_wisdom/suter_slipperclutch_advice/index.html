Hello guys, I've got the chance to buy a suter slipperclutch. Used for approx 2000miles in a 08 sdr (the one with more power + rotty box, akra's etc) and I think he's running the 1325Nm spring
After doing a bit of searching on here, I found a couple threads of people having issues, slipping for bic and kol. Klinck, wear on drswade's racebike etc.
But no real solid posts in those thread of the actual solving of the problems. Drswade sold his and went for a different clutch. Klinck has long ago sold his bike. So that leaves Bic: how is your clutch doing? Did you have any more issues with clutch slipping?
And should I go for the suter? The current owner has been having no problems at all, so the plan would be to just start on his settings (springs) and see from there. I've got a non R sd, with motobox, fmf apex etc, so if the clutch can handle the current bikes power, it should be able to handle mine right?
I only ride on the roads, and the rear does occasionaly lock up on downshifts, but nothing serious.
I've no experience with slipper clutches so far, so it wouldn't be much more than a gimmick.

Nampus mate.

Since the original, (and only) problem me and Klinky experienced I've had no issues at all with the Suter clutch. Been brilliant. I think we both had the same issue with slipping because the spring rate was a bit low for the additional power we both saw from fitting motoboxes and full Akra systems. If you've found the thread you'll have seen what spring rate we changed to, (I've forgotten) but it cured the problem in both cases.

The service from Suter was very good when I needed to ask questions.

Anyway, I'd certainly recommend getting one if you have the chance at a good price - you'll never want to go back to riding without a slipper!

Let me know if you have any other questions.
