Hold off with the SAS removal kit for now.
use Tuneecu to load another map to boost midange, the akra map is waaaay too rich. If you just run the other cans use guys map out of the how to download section. just tick the second butterflies back on before you load the map as I had it switched off as I took the 2nd butterflies out.

This map will greatly boost power and torque compared to the standard and akra maps, it also has the 2-3 rd gear limiter disabled and sas switched off.
If you like it you can fit the SAS removal kit if you want to get rid of the tubing etc.

Save your map before uploading the new one so you can revert back if you want.

2011-03-10 10:17:00 UTC

Turning SAS of in TuneECU is sufficient. If you don't like all the tubes you can remove them and mount the plates.
When you use Guys map load it in TuneECU but before you send the map to the bike enable 2nd throttle.

So step by step.
Connect to your bike with TuneECU.
Read the current map from your bike.
Save it somewhere on you PC (something like My_original_map)
Load Guys map in TuneECU
Enable 2nd throttle by clicking the checkbox.
Send Guys map to your bike
