I have fitted an SDR seat unit to my '07 SD and installed the KTM alarm that I had since new. There is a remote cable and plastic contact break terminals which I've always presumed was so that if the seat was removed the alarm would go off. I thought that it works with some sort of magnetic connection or something that activates the alarm when they are pulled apart - there is no physical connection but the surfaces have worn away so they must touch each other. I never actually tested it on the dual seat so have no idea if it ever actually worked.

There was nothing in the single seat instructions about where to fit this so I made the best judgement - see pictures. There seemed to be two places that they wanted to go - in the top of the under-tray in a recess and onto the bottom of the seat unit where there were two really small indents that corresponded to the spacing of the holes in the terminal - but I have no idea if they actually connect up when the seat is in place. All I do know is that when the bike is alarmed and I take off the seat it does not set the alarm off. The alarm does work with the motion sensor so it's wired up and working OK.

Has anyone with an SDR got an alarm fitted and does it look like I have the contact break terminals screwed into the right place and the right way round?

Any help appreciated.
