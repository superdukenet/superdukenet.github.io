I'm going to rewire my indicators /turn signals + licence plate light this week. I had decided to use the Rizoma mainly because I hate the KTM licence plate light. So last night I decided ok, never mind trying to keep the KTM wiring so the original parts can go back, I'm going to extend the wires of the mini indicators so they run up into the underseat tray and have the connectors there, to connect to the KTM 4-wire connector.

oh, and this only because I got a smallerz - illegal - number plate. Our legal ones are exactly the same size as yours, I think. The one I had made is 170x150.
