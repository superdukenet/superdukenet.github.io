Hiya - the map I'm using is the KTM map for Akras (KM611EU0802031), so I got my dealer to load it.
Tuneboy can also load this map (something which PCIII can't) - goto their UK website & download it: http://www.tuneboy.co.uk/model/80/KTM/9 ... DukeR.aspx

Re SAS - it's fitted on all KTM bikes (& most others under various names), so did you buy 2nd hand then & previous owner disable it ?

As far as using tuneboy to disable the O2 sensors, it was possible with the 05/06 SD by loading a specific basemap (which disabled them).
However, none of the 08 SDR basemaps disable the 02 sensors, so you'll have to revert to the physical method - ie: PCIII 02 eliminators, or equivalent.
